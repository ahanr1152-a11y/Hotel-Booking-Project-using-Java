# Hotel-Booking-Project-using-Java

A simple console-based Hotel Room Booking System developed using Java. This project allows users to view room availability, book rooms, and cancel bookings through a menu-driven interface.

## 📌 Features

- View available and booked rooms
- Book a room
- Cancel a room booking
- Input validation for room numbers
- Menu-driven console application

## 🛠️ Technologies Used

- Java
- Scanner Class
- Arrays
- Methods
- Loops
- Conditional Statements
- Switch Case

## 📂 Project Structure

```
HotelBooking.java
```

## 🚀 How to Run

1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile the program:
   ```
   javac HotelBooking.java
   ```
4. Run the program:
   ```
   java HotelBooking
   ```
